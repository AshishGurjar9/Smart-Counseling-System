# Smart Student Counseling System

## 📌 Project Overview
This is a Java-based console application that helps students choose the right career path based on their skills and interests. The system provides guidance and suggestions to students for better career decision making.

---

## 🚀 Features
- Career Matching based on skills
- Skill Development Suggestions
- Job Market Trends information
- Success Stories for motivation
- Simple and user-friendly console interface

---

## 💻 Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class (Input Handling)

---

## 🎯 Purpose of Project
The main purpose of this project is to guide students in selecting the right career path according to their interests and skills.
